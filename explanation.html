explanation

context

Uncertainty Principle says that we cannot measure the position and velocity (ignoring mass in momentum gives velocity) of a subatomic particles, say electron simultaneously. Either we can measure position or velocity one at a time with more accuracy or we can try to measure both at very poor accuracy.

There are two ways to solve this.

  • We can try to increase the accuracy of both position and velocity while measuring at single go. Problem with this is, it is practically not possible yet.
  • We can try to measure position and velocity seperatly*. Measuring twice is wastage of resources and not efficient so we do not want that.

consistency vs uncertainty

We have two operations,

  1. Either we can have both position and velocity with lesser accuracy
  2. Or we can have more accuracy and with only one of the two.

Upon examining the two operations there are two features (i.e. parameters),

  • X - position
  • Y - velocity
and one weight (i.e. magnitude),
  • W - accuracy

If we choose operation 1 we will get {W1X, W2Y}. That is, we will get both of the features with poor weights consistantly.
If we choose operation 2 we will get {W<X|Y>}. That is, we will get the best weight but there will be an uncertainty on which feature we will get.
